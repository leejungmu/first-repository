※로컬저장소와 원격저장소 연결
// Github 원격저장소와 연결한다.
git remote add origin [자신의 Github 원격저장소 주소] 
// 연결된 원격저장소 확인한다.
git remote -v


※ git clone ▶▶로컬저장소에 원격저장소 생성 
> git pull ▶▶fetch(다운로드만 하고 저장 X) + merge
> git pull --rebase ▶▶fetch + rebase 
                               히스토리가 엇갈려 애매모호한 상황에서 작업을 원격저장소의 최신 상태와 합치기
